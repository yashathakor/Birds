How to deploy
=============
1. Extract contents of BirdTeamProject.zip
2. Copy the contents of BirdTeamDatabase.SQL to reconstruct database
3. Copy the contents of the Website Files folder to the htdocs folder
4. Enter the localhost address into URL bar
5. Enjoy the birds
